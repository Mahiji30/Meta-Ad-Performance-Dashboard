# Meta-Ad-Performance-Dashboard

### Power BI | Data Analysis | Visualization | DAX | Meta Ads Insights  

An interactive **Power BI dashboard** designed to analyze and visualize advertising performance from **Meta (Facebook & Instagram)** campaigns. The dashboard provides deep insights into campaign efficiency through key metrics such as **CTR, CPC, ROAS, Conversion Rate, and Spend Distribution** ‚Äî empowering marketers and data analysts to make data-driven optimization decisions.  

---

## Project Overview  

Advertising performance data can often be overwhelming when viewed in raw form.  
This project simplifies that by transforming exported Meta Ads data into **visually appealing, actionable dashboards** with interactive filtering and automated KPI computation.  

**Key objectives:**  
- Consolidate campaign-level ad performance metrics.  
- Track and visualize trends, engagement, and cost efficiency.  
- Identify underperforming ads and optimize ROI.  
- Provide one-click filtering and drillthrough for campaign insights.  

---

## Features  

**Interactive Visualizations** ‚Äì Campaign, ad set, and time-based filters for flexible analysis.  
**Comprehensive Metrics** ‚Äì Impressions, Clicks, CTR, CPC, CPM, Conversions, ROAS, Spend, and Cost per Result.  
**Anomaly Detection** ‚Äì Highlights unusual fluctuations in CTR or conversion rates.  
**Dynamic KPIs** ‚Äì Real-time calculation of key ratios using DAX.  
**Automated Reporting** ‚Äì Reduces manual Excel reporting time and improves decision-making speed.
