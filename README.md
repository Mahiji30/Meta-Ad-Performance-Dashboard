## üöÄ Project Overview  

Advertising performance data can often be overwhelming when viewed in raw form.  
This project simplifies that by transforming exported Meta Ads data into **visually appealing, actionable dashboards** with interactive filtering and automated KPI computation.  

**Key objectives:**  
- Consolidate campaign-level ad performance metrics.  
- Track and visualize trends, engagement, and cost efficiency.  
- Identify underperforming ads and optimize ROI.  
- Provide one-click filtering and drillthrough for campaign insights.
